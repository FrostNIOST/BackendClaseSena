## NPM

* npm install express@5.1.0 
* npm install mongoose@8.15.1
* npm install bcryptjs@2.4.3 para encriptar
* npm install jsonwebtoken@9.0.2
* npm install cors para conectar front y back 
* npm install dotenv@16.4.7 el que le da variables de entorno para rutas con la base de datos 
* npm install --save-dev nodemon@3.1.9 para arrancar el server sin detenerlo (se refresca solito)

* *NOTA* npm install para volver a descargar las dependencias (cuando se pone gitignore de los modulos)
